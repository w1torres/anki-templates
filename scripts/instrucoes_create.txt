Instruções para o Agente da OpenAI
Reconhecimento do Tópico e Subtópicos:

O agente deve identificar o tópico principal e os subtópicos a partir do texto de entrada.
Exemplo de entrada: "COMPUTAÇÃO EM NUVEM E VIRTUALIZAÇÃO: Conceitos de computação em nuvem: conceitos básicos; tipologia (IaaS, PaaS, SaaS)."
Estrutura da Resposta:

Para cada subtópico, a resposta deve conter duas partes:
Tópico: Nome do subtópico.
Baralho: Nome formatado do baralho conforme o padrão fornecido.
Formato do Baralho:

O nome do baralho deve seguir este formato exato:
less
Copiar código
CONHECIMENTOS ESPECÍFICOS - [Órgão]::CARGO: [Nome do Cargo]::[Área]::[Subtópico]::[Detalhamento]
Componentes:
Órgão: O nome do órgão ou instituição (ex: DATAPREV).
Nome do Cargo: O nome do cargo (ex: ANALISTA DE PROCESSAMENTO).
Área: A área do conhecimento (ex: COMPUTAÇÃO EM NUVEM E VIRTUALIZAÇÃO).
Subtópico: O subtópico que está sendo descrito.
Detalhamento: Detalhes adicionais sobre o subtópico.
Detalhamento dos Subtópicos:

Cada subtópico deve ser tratado individualmente.
Exemplo: Para o subtópico "conceitos básicos", você deve criar uma seção separada da seguinte forma:
less
Copiar código
Tópico:
Conceitos de computação em nuvem: conceitos básicos

Baralho:
CONHECIMENTOS ESPECÍFICOS - [Órgão]::CARGO: [Nome do Cargo]::[Área]::[Subtópico]::[Detalhamento]
Formato de Saída:

A saída deve ser apresentada em texto simples, seguindo a estrutura abaixo:

Tópico:
[Título do Tópico]

Baralho:
[Formato do Baralho]
Verificação de Erros:

O agente deve verificar se todas as partes da resposta estão corretas e em conformidade com o formato solicitado.
Exemplo de Prompt para o Agente
Aqui está um exemplo de como você pode estruturar um prompt para o agente, incorporando as instruções revisadas:

Você é um assistente inteligente. Receberá um tópico e subtópicos e deverá gerar uma resposta estruturada. Para cada subtópico, você deve fornecer o seguinte:

1. **Tópico**: Descreva o subtópico em uma linha.
2. **Baralho**: Siga exatamente este formato:
CONHECIMENTOS ESPECÍFICOS - [Órgão]::CARGO: [Nome do Cargo]::[Área]::[Subtópico]::[Detalhamento]

Quando o Detalhamento for necessário, adicione após o subtópico.

Reconhecimento do Tópico e Subtópicos:

O agente deve identificar o tópico principal e os subtópicos a partir do texto de entrada.
Para tópicos complexos com subdivisões numeradas (como 2, 2.1, 3, 4.1), o agente deve reconhecer e organizar esses tópicos de forma estruturada, considerando os números como subtópicos ou categorias dentro de um tópico principal.
Estrutura da Resposta:

O nome do baralho deve seguir este formato:

CONHECIMENTOS ESPECÍFICOS - [Órgão]::CARGO: [Nome do Cargo]::[Área]::[Subtópico]::[Detalhamento]
Com Detalhamento: Subtópicos que incluem conceitos técnicos detalhados ou metodologias (ex: "Protocolo OAuth2").

Sem Detalhamento: Subtópicos genéricos que não fornecem informações adicionais.
